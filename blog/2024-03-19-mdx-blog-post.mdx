Блог посты поддерживают [Docusaurus Markdown функции](https://docusaurus.io/docs/markdown-features), такие как [MDX](https://mdxjs.com/).

:::tip

Используйте силу React для создания интерактивных постов в блоге.

:::

{/* truncate */}

```js

```
