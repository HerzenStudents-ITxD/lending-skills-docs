# React Hooks: Полное руководство для начинающих

> "Хуки — это способ использовать состояние и другие возможности React без написания классов."

React Hooks появились в версии 16.8 и полностью изменили подход к написанию компонентов в React.

<!-- truncate -->

## Основные хуки

### useState

```jsx
function Счетчик() {
  const [счетчик, установитьСчетчик] = useState(0);
  
  return (
    <div>
      <p>Счётчик: {счетчик}</p>
      <button onClick={() => установитьСчетчик(счетчик + 1)}>
        Увеличить
      </button>
    </div>
  );
}
```

### useEffect

```jsx
function ПолучениеДанных() {
  const [данные, установитьДанные] = useState(null);
  
  useEffect(() => {
    fetch('https://api.example.com/data')
      .then(ответ => ответ.json())
      .then(данные => установитьДанные(данные));
  }, []); // Пустой массив зависимостей = только при монтировании
  
  return <div>{данные ? JSON.stringify(данные) : 'Загрузка...'}</div>;
}
```

## Лучшие практики

1. Используйте хуки только на верхнем уровне
2. Не вызывайте хуки внутри условий
3. Следите за зависимостями в useEffect
4. Создавайте кастомные хуки для переиспользуемой логики

## Пример кастомного хука

```jsx
function useРазмерОкна() {
  const [размер, установитьРазмер] = useState({
    ширина: window.innerWidth,
    высота: window.innerHeight
  });

  useEffect(() => {
    const обработчикИзмененияРазмера = () => {
      установитьРазмер({
        ширина: window.innerWidth,
        высота: window.innerHeight
      });
    };

    window.addEventListener('resize', обработчикИзмененияРазмера);
    return () => window.removeEventListener('resize', обработчикИзмененияРазмера);
  }, []);

  return размер;
}
```

## Заключение

React Hooks значительно упрощают работу с состоянием и побочными эффектами в функциональных компонентах. Они делают код более чистым и понятным, а также облегчают переиспользование логики между компонентами.

> Изучайте хуки, экспериментируйте и создавайте более чистый и поддерживаемый код!
